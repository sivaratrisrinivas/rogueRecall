Third-party reference excerpts, Rights Evidence, and Target System responses

Record and in the Release Notice Bundle. Inclusion does not imply endorsement.

Report a rights concern through a private maintainer contact when disclosure
would expose personal data or restricted Rights Evidence, or through a GitHub
issue for public concerns. Preserve the affected Benchmark Corpus Release and
Run Records while the concern is reviewed; suspension, withdrawal, and
replacement occur through signed append-only Corpus Status Records and never
by silently changing an existing release.
